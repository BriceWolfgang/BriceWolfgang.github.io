Lets put the top five family names in as dummies

In [46]:
# get family name from Name column
df['family_name'] = [x.split(',')[0] for x in df['Name']]

Top5Fam = df['family_name'].value_counts()[:5].index # Store the list of names

# make a column in the data frame for each value, and fill it with the dummy info
for fam_name in Top5Fam:
    df[fam_name] = [x == fam_name for x in df['family_name']]
In [25]:
df.dropna(inplace=True) # Sns pairplot does not like NaN

Immediately you can see outliers in age and fair or perhaps a row with fair and age switched? But what about categorical data?
Lets filter the data down to the five most common family names and see how they faired.

In [ ]:
# Fun fact, if a list comprehension is the same length as a data frame
# you can use it as a filter.

dfT5 = df[[x in Top5Fam for x in df['family_name']]]
In [48]:
# But we both know my love of list comprehensions is getting out of hand
# so here is the pythonic? pandonic? way to filter a column for specific values

dfT5 = df[df['family_name'].isin(Top5Fam)] # more readable for sure

Here we can see the breakdown of Family names by ticket price and survival.

In [55]:
sns.swarmplot(x="family_name", y="Fare",hue="Survived", data=dfT5)
